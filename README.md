# Hotel Review Sentiment Analysis

This project is a Machine Learning and NLP-based web application that predicts whether a hotel review is Positive or Negative.

## Project Overview

The project uses:
- Natural Language Processing (NLP)
- TF-IDF Vectorization
- Logistic Regression
- Streamlit Web Application

The model is trained on hotel review data and classifies user-entered reviews into positive or negative sentiment.

---

## Features

- Text preprocessing using NLTK
- Stopword removal and stemming
- TF-IDF vectorization
- Logistic Regression model
- Real-time prediction using Streamlit
- Interactive user interface

---

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- NLTK
- Streamlit
- Matplotlib
- Seaborn

---

## Model Performance

| Model | Accuracy |
|------|------|
| Logistic Regression | 87.9% |
| Naive Bayes | 86.5% |

Logistic Regression performed better and was selected as the final model.
